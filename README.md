CLI Calculator README
Overview:
This calculator does math using a Python app. It handles tasks like adding or subtracting numbers. Multiplication and division also work inside it. When someone types an operation, the result shows up right away. Mistakes get caught before they cause problems. You can keep using it without stopping.

Features:
Interactive menu-driven interface

Handles decimal numbers, including those with fractions

Division by zero protection

Input validation for operators and numbers

Keep going until they enter 'quit'.

Prerequisites:
Installed Python 3.x. Ready to go.

VS Code or any text editor

Terminal/Command Prompt

Installation:
Save the code inside a file named calculator.py

Open terminal in the file directory

Run: python calculator.py
